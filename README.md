# AI FundBridge Platform Package

AI FundBridge is a concrete platform design and MVP prototype for helping the public apply for AI innovation grants, accelerators, compute credits, fiscal-hosted funding, and early-stage financing.

## What is included

- `docs/AI_FundBridge_Platform_Blueprint.docx` — full Chinese platform blueprint.
- `docs/AI_FundBridge_PRD_Technical_Spec.docx` — PRD, technical architecture, API and data model.
- `docs/AI_FundBridge_DIKWP_Governance_Manual.docx` — DIKWP audit and governance manual.
- `docs/AI_FundBridge_Template_Library_Handbook.docx` — grant/investor templates and operating handbook.
- `sheets/AI_FundBridge_Funding_Database_and_Scoring.xlsx` — funding database, scoring, intake, budget and evidence ledgers.
- `prototype/index.html` — offline browser prototype.
- `api/fundbridge_openapi.yaml` — initial API schema.
- `templates/` — application canvas, investor one-pager, evidence ledger CSV and email templates.
- `sources/source_ledger.md` — source list for program data.

## How to run the prototype

Open `prototype/index.html` in a modern browser. No server, login, network or model API is required.

## Core principle

Do not help applicants fake fundability. Help them find fit, repair gaps and build evidence-linked application packs.
